myFlix API Documentation

API that supplies information for webapp myFlix.

Information supplies includes: movies, user info, genre info.

Allows new users to register /users POST A JSON object holding information about the new user (username).

{

"name": "Mary",

"favoriteMovies": [],

}

 A JSON object holding data (username, id).

"name": "Mary",

"favoriteMovies": [],

}
Allows users to update their username info (username) /users/:id PUT A JSON object containing data about the new username.

{

"name": "New Name",

"favoriteMovies": []

}

 A JSON object containing data about the user including id, and new username.

{

"id": 1,

"name": "New Name",

"favoriteMovies": []

}
Allow users to add a movie to their list of favorites /users/:id/:movieTitle POST None A message indicating if the movie was successfuly added to the user's favourites array.
Allow users to remove a movie from their list of favorites /users/:id/:movieTitle DELETE None A message indicating if the movie was successfuly deleted from the user's favourites array.
Allow existing users to deregister /users/:id DELETE None A message indicating if the user was successfuly deleted from the users array.
