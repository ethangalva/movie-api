myFlix API Documentation

API that supplies information for webapp myFlix.

Information supplies includes: movies, user info, genre info.

Allows new users to register /users POST A JSON object holding information about the new user.

{

"Username": "Username",

"Password": "Password",

"Email": "email@gmail.com",

"Birthday": "2000-01-01",

"FavoriteMovies": [],

}

 A JSON object holding data (username, password, email, birthday, favoriteMovies, ID).

{

"Username": "Username",

"Password": "Password",

"Email": "email@gmail.com",

"Birthday": "2000-01-01",

"FavoriteMovies": [],

"_id": "62f69064b9d006366cb9640f"

}
Allows users to update their username by their id /users/:_id PUT A JSON object containing data about the new username.

{

"Username": "new Username"

}

 A JSON object containing data about the user including id, and new username.

{

"Username": "new Username",

"Password": "Password",

"Email": "email@gmail.com",

"Birthday": "2000-01-01",

"FavoriteMovies": [],

"_id": "62f69064b9d006366cb9640f"

}
Allow users to add a movie to their list of favorites /users/:_id/favoriteMovies/:movieID POST None A message indicating if the movie was successfuly added to the user's favourites array.
Allow users to remove a movie from their list of favorites /users/:_id/favoriteMovies/:movieID DELETE None A message indicating if the movie was successfuly deleted from the user's favourites array.
Allow existing users to deregister /users/:Username DELETE None A message indicating if the user was successfuly deleted from the users array.
